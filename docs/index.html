入力条件

  • 毎月の積立額: 整数 / 範囲: 0円〜999,999円 / ボタン: ±¥10,000
  • 想定年利回り: 整数 / 範囲: 0%〜99% / ボタン: ± 1%
  • 運用期間: 5年、10年、15年、25年、30年から選択

試算結果

  • 「合計資産 = 毎月の積立額 × 12ヶ月 × (1 + 想定年利回り)^運用期間」で算出しています。
  • 実際の資産運用では、税金、手数料等が差し引かれる為、試算結果と実際の資産推移は異なります。
  • 為替の変動、利回りの変動は考慮されていません。
